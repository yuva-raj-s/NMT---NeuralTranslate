What is Neural Machine Translation?

Neural Machine Translation (NMT) is an advanced approach that uses artificial neural networks to translate text from one language to another. Unlike traditional methods, NMT considers the entire sentence context to produce more natural and accurate translations.

The system learns patterns from millions of translated texts, enabling it to understand language nuances that traditional rule-based systems cannot capture.

About mBART

mBART (Multilingual BART) is a state-of-the-art sequence-to-sequence model pre-trained on large-scale monolingual corpora in many languages. It's specifically designed to handle translation between multiple languages.

As one of the first methods to pre-train a complete sequence-to-sequence model by denoising full texts in multiple languages, mBART enables high-quality translations across diverse language pairs.

Supported Languages

This application supports translation from English to all 13 languages available in the ALT (Asian Language Treebank) dataset:

  • Bengali (bn)
  • English (en)
  • Filipino (fil)
  • Hindi (hi)
  • Bahasa Indonesia (id)
  • Japanese (ja)
  • Khmer (km)
  • Lao (lo)
  • Malay (ms)
  • Myanmar (my)
  • Thai (th)
  • Vietnamese (vi)
  • Chinese (zh)

About the ALT Dataset

The ALT (Asian Language Treebank) dataset is a high-quality parallel corpus covering numerous Asian languages. It contains professional translations of the same content across multiple languages, providing a reliable foundation for training neural machine translation models.

This dataset is particularly valuable for low-resource Asian languages, which often have limited training data available. By using the ALT dataset, our translation system can provide accurate translations even for language pairs that are typically challenging for machine translation.
