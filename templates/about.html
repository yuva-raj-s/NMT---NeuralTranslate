About This Application

Neural Machine Translation App

This web application provides neural machine translation from English to various Asian languages using mBART (Multilingual BART) pre-trained model.

Technology Stack

  • Backend: Python, Flask, Hugging Face Transformers
  • Model: mBART (facebook/mbart-large-50-many-to-many-mmt)
  • Frontend: HTML5, Bootstrap 5, JavaScript

About mBART

mBART is a sequence-to-sequence denoising auto-encoder pre-trained on large-scale monolingual corpora in many languages using the BART objective. mBART is one of the first methods for pre-training a complete sequence-to-sequence model by denoising full texts in multiple languages, while previous approaches have focused only on the encoder, decoder, or reconstructing parts of the text.

Supported Languages

This application supports translation from English to all 13 languages available in the ALT (Asian Language Treebank) dataset:

  • Bengali
  • English
  • Filipino
  • Hindi
  • Bahasa Indonesia
  • Japanese
  • Khmer
  • Lao
  • Malay
  • Myanmar (Burmese)
  • Thai
  • Vietnamese
  • Chinese (Simplified)

About the ALT Dataset

The ALT (Asian Language Treebank) dataset is a parallel corpus covering several Asian languages. It contains translations of the same content across multiple languages, making it ideal for training and fine-tuning neural machine translation models, especially for low-resource Asian languages.
